# API

## `newTab`
Create a new tab

```javascript
const tab = this.$tabs.newTab()

// or provide tab options
this.$tabs.newTab({ ... })
```

## `updateTab`
Update and existing tab

```javascript
this.$tabs.updateTab(tab, data)
```

## `removeTab`
Remove a tab

```javascript
this.$tabs.removeTab(tab)
```

## `getTabs`
List all tabs

```javascript
this.$tabs.getTabs(tab)
```

## `to`
Route to a tab

```javascript
this.$tabs.to({ ... })
```
